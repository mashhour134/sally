# دليل تحويل تطبيق "حارس الهواء" إلى تطبيق جوال (iOS & Android)
## Mobile Deployment Guide for "Air Guard"

لقد قمنا بتهيئة وتثبيت حزمة **Capacitor** (بواسطة Ionic) في كود المشروع الخاص بك. هذه الحزمة تتيح لك تشغيل كود الـ React & Vite الخاص بك كتطبيق جوال حقيقي وأصلي (Native iOS & Android) بنفس الميزات والواجهات المصممة بدقة.

يمكنك تنزيل الكود بالكامل كملف مضغوط **ZIP** أو ربطه بـ **GitHub** من خلال زر الإعدادات في أعلى واجهة AI Studio، ثم اتباع الخطوات أدناه على جهازك الشخصي.

---

## 🛠️ المتطلبات الأساسية (Prerequisites)

قبل البدء بالتحزيم على جهازك الشخصي، تأكد من تثبيتها:
1. **Node.js** (إصدار 18 أو أحدث).
2. لـ **Android**: قم بتثبيت **Android Studio** وتجهيز محاكي (Emulator) أو هاتف أندرويد حقيقي عبر تصحيح أخطاء الـ USB (USB Debugging).
3. لـ **iOS**: يتطلب جهاز **Mac (macOS)** وتثبيت مجمع **Xcode** ومحاكي آيفون.

---

## 🚀 خطوات بناء التطبيق وتشغيله (Step-by-Step Guide)

افتح سطر الأوامر (Terminal) في مجلد المشروع الذي قمت بتنزيله، ونفّذ الخطوات التالية:

### 1️⃣ تثبيت الاعتماديات الأساسية (Install Dependencies)
```bash
npm install
```

### 2️⃣ بناء نسخة الويب المستقرة (Build the Web Application)
تأكد من تجميع ملفات الويب داخل مجلد `dist` أولاً:
```bash
npm run build
```

---

## 🤖 أولاً: التشغيل والتحويل لنظام الأندرويد (Android)

### 3️⃣ إضافة بيئة عمل الأندرويد للمشروع (Add Android Platform)
```bash
npm run mobile:add-android
```
*(هذا الأمر سينشئ مجلد `android` أصلي ومستقل يحتوي على مشروع أندرويد جاهز للفتح في Android Studio).*

### 4️⃣ مزامنة الكود وتحديث الواجهات (Syncing the Code)
عند إجراء أي تعديل مستقبلاً على كود الـ React، نفّذ بناء الويب أولاً ثم قم بمزامنة الملفات مع الأندرويد باستخدام:
```bash
npm run mobile:sync
```

### 5️⃣ فتح وترقية التطبيق عبر Android Studio (Open & Run in Android Studio)
لفتح المشروع تلقائياً في بيئة تطوير أندرويد وبناء ملف الـ `APK` النهائي:
```bash
npm run mobile:open-android
```
*أو يمكنك فتح مجلد `android` داخل تطبيق Android Studio يدوياً.*
- بمجرد اكتمال الفتح والمزامنة البرمجية (Gradle Sync) داخل Android Studio، اضغط على زر **Run** لتشغيل التطبيق على محاكي أو هاتفك الشخصي.
- لإنتاج ملف التثبيت النهائي، اذهب إلى `Build` -> `Build Bundle(s) / APK(s)` -> `Build APK(s)`.

---

## 🍎 ثانياً: التشغيل والتحويل لنظام الآيفون (iOS)
*(ملاحظة: يتطلب نظام macOS وبرنامج Xcode)*

### 3️⃣ إضافة بيئة عمل الآيفون للمشروع (Add iOS Platform)
```bash
npm run mobile:add-ios
```

### 4️⃣ مزامنة الملفات للآيفون (Syncing the Code)
```bash
npm run mobile:sync
```

### 5️⃣ فتح وترقية التطبيق عبر Xcode (Open & Run in Xcode)
لفتح المشروع تلقائياً في Xcode:
```bash
npm run mobile:open-ios
```
- اختر المحاكي المستهدف (Simulator) أو جهاز الآيفون الخاص بك والمربوط بالكمبيوتر.
- اضغط على زر **Play / Run** لبناء وتشغيل التطبيق على الهاتف مباشرة.

---

## 💡 نصائح وحلول سريعة (Tips & Troubleshooting)

*   **تحديث الواجهات:** تذكر دائماً أنه بعد تعديل أي سطر برمجي في React، يجب تشغيل:
    ```bash
    npm run build
    npm run mobile:sync
    ```
    لتمرير التحديثات لملفات التطبيق داخل مجلدات Android و iOS.
*   **عناوين الـ API:** تطبيق "حارس الهواء" يستخدم خادماً داخلياً (Express Node Server) للمحاكاة والمستشعرات. عند تجميع التطبيق للعمل على الجوال الحقيقي بشكل مستقل، تأكد من تحديث طلبات الـ HTTP (fetch) لتوجه إلى رابط الخادم الفعلي (Hosted URL) الخاص بك بدلاً من الروابط النسبية المقيدة مثل `/api/devices`.
